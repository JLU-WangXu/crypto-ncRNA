<h3 id="pageName">CAVP Testing: Block Cipher Modes</h3>


<p id="content">
    <h4><a id="Top" name="Top"></a>Algorithm Specifications</h4>

<p>Algorithm specifications for current FIPS-approved and NIST-recommended block cipher modes are available from the&nbsp;<a data-csrc-link="true" data-node-guid="54053d32-2050-4532-b610-50a6395cc99b" href="/projects/block-cipher-techniques">Cryptographic Toolkit</a>.</p>

<p>Current testing includes the following block cipher modes:</p>

						</td>
					</tr>
					<tr>
						<td>
						<h6><a href="#CCM">CCM</a>&nbsp;(SP 800-38C)</h6>
						</td>
						<td>
						<h6><a href="#KW">KW / KWP / TKW</a>&nbsp;(SP 800-38F)(Key Wrap using AES and Triple-DES)</h6>
						</td>
					</tr>
					<tr>
						<td>
						<h6><a href="#GCMVS">GCM / GMAC / XPN</a>&nbsp;(SP 800-38D and CMVP Annex A)</h6>

<p><sub>For testing of&nbsp;ECB&nbsp;(Electronic Codebook),&nbsp;CBC&nbsp;(Cipher Block Chaining),&nbsp;OFB&nbsp;(Output Feedback),CFB&nbsp;(Cipher Feedback) and&nbsp;CTR&nbsp;(Counter) modes from SP 800-38A, see the&nbsp;<a data-csrc-link="true" data-node-guid="838d0e73-d331-4ca2-8208-a3ebfde65599" href="/projects/cryptographic-algorithm-validation-program/block-ciphers">CAVP block ciphers page</a>.</sub></p>

<hr>
<h4><a id="Testing" name="Testing"></a>Algorithm Validation Testing Requirements</h4>

<p><em><a id="CMAC" name="CMAC"></a>CMAC: Block Cipher-based Message Authentication Code</em></p>

<div style="background:#eee;border:1px solid #ccc;padding:5px 10px;">
<p>The&nbsp;<a href="/CSRC/media/Projects/Cryptographic-Algorithm-Validation-Program/documents/mac/CMACVS.pdf">CMAC Validation System (CMACVS)</a>&nbsp;specifies validation testing requirements for the CMAC mode in SP 800-38B.</p>

<p><strong>Testing Notes</strong></p>

<ul>
	<li>Prerequisites for CMAC testing are listed in the&nbsp;<a href="/CSRC/media/Projects/Cryptographic-Algorithm-Validation-Program/documents/CAVPFAQ.pdf">CAVP Frequently Asked Questions (CAVP FAQ)</a>&nbsp;General Question GEN.5.</li>
</ul>

<ul>
	<li>As of 1-1-2016, TDES KO2 encrypt is no longer compliant. TDES KO2 decrypt is allowed for legacy use only. (See SP800-131A Revision 1.)</li>
</ul>
</div>

<p>&nbsp;</p>

<p><em><a id="CCM" name="CCM"></a>CCM: Counter with Cipher Block Chaining Message Authentication Code</em></p>

<div style="background:#eee;border:1px solid #ccc;padding:5px 10px;">
<p>The&nbsp;<a href="/CSRC/media/Projects/Cryptographic-Algorithm-Validation-Program/documents/mac/CCMVS.pdf">CCM Validation System (CCMVS)</a>&nbsp;specifies validation testing requirements for the CCM mode in SP 800-38C.</p>

<p>Testing Notes</p>

<ul>
	<li>Prerequisites for CCM testing are listed in the&nbsp;<a href="/CSRC/media/Projects/Cryptographic-Algorithm-Validation-Program/documents/CAVPFAQ.pdf">CAVP Frequently Asked Questions (CAVP FAQ)</a>&nbsp;General Question GEN.5.</li>
</ul>

<ul>
	<li>As of 1-1-2016, TDES KO2 encrypt is no longer compliant. TDES KO2 decrypt is allowed for legacy use only. (See SP800-131A Revision 1.)</li>
</ul>
</div>

<ul>
</ul>

<p><em><a id="GCMVS" name="GCMVS"></a>GCM, GMAC, XPN: Galois/Counter Mode, GCM Message Authentication Code, and GCM-AES-XPN mode</em></p>

<div style="background:#eee;border:1px solid #ccc;padding:5px 10px;">
<p>The&nbsp;<a href="/CSRC/media/Projects/Cryptographic-Algorithm-Validation-Program/documents/mac/gcmvs.pdf">GCM, GMAC and XPN Validation System (GCMVS)</a>&nbsp;specifies validation testing requirements for the GCM and GMAC modes in SP 800-38D and GCM-AES-XPN mode from IEEE Std 802.1AEbw-2013 (See CMVP Annex A).</p>

<p>Testing Notes</p>

<ul>
	<li>Prerequisites for GCM, GMAC, and XPN testing are listed in the&nbsp;<a href="/CSRC/media/Projects/Cryptographic-Algorithm-Validation-Program/documents/CAVPFAQ.pdf">CAVP Frequently Asked Questions (CAVP FAQ)</a>&nbsp;General Question GEN.5.</li>
</ul>
</div>

<ul>
</ul>

<p><em><a id="XTS" name="XTS"></a>XTS-AES</em></p>

<div style="background:#eee;border:1px solid #ccc;padding:5px 10px;">
<p>The&nbsp;<a href="/CSRC/media/Projects/Cryptographic-Algorithm-Validation-Program/documents/aes/XTSVS.pdf">XTS-AES Validation System (XTSVS)</a>&nbsp;specifies validation testing requirements for the XTS-AES mode in SP 800-38E.</p>

<p>Testing Notes</p>

<ul>
	<li>Prerequisites for XTS-AES testing are listed in the&nbsp;<a href="/CSRC/media/Projects/Cryptographic-Algorithm-Validation-Program/documents/CAVPFAQ.pdf">CAVP Frequently Asked Questions (CAVP FAQ)</a>&nbsp;General Question GEN.5.</li>
</ul>
</div>

<ul>
</ul>

<p><em><a id="KW" name="KW"></a>KW, KWP and TKW: Key Wrapping and Authenticated Encryption and Decryption</em></p>

<div style="background:#eee;border:1px solid #ccc;padding:5px 10px;">
<p>The&nbsp;<a href="/CSRC/media/Projects/Cryptographic-Algorithm-Validation-Program/documents/mac/KWVS.pdf">Key Wrap Validation System (KWVS)</a>&nbsp;specifies validation testing requirements for the AES Key Wrap (KW), AES Key Wrap with Padding (KWP) and Triple DEA Key Wrap (TKW) modes in SP 800-38F.</p>

<p>Testing Notes</p>

<ul>
	<li>Prerequisites for&nbsp;KW, KWP, and TKW&nbsp;testing are listed in the&nbsp;<a href="/CSRC/media/Projects/Cryptographic-Algorithm-Validation-Program/documents/CAVPFAQ.pdf">CAVP Frequently Asked Questions (CAVP FAQ)</a>&nbsp;General Question GEN.5.</li>
</ul>
</div>
